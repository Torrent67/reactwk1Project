## Setup/Installation Requirements

1. Clone this repository:
    ```
    $ git clone 
    ```
2. Install dependencies:
    ```
    $ npm install
    ```
3. Build distribution files:
    ```
    $ webpack
    ```
4. Run server :
    ```
    $ npm run start
    ```
5. Open Site :
    ```
    http://localhost:8080/
    ```
